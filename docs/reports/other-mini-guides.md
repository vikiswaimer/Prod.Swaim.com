# Отчёт: Дополнительно — мини-гайды (как Цифровая безопасность)

Дата: 2026-07-23 · Ветка: `cursor/other-mini-guides-7cdb`

## Цель

Группа **Дополнительно / Other** — не просто «складывай сущности», а **мини-советы под название блока**: зачем, что даёт бизнесу, как вести, какими инструментами. Эталон — уже сделанная «Цифровая безопасность».

Пакеты E–J (roadmap v2) не трогали: это отдельный срез UX раздела Other.

### Демо-записи

- `Alice` — карточка-демо с полями сигнала (не выдуманный «реальный» отзыв без пометки).
- Email / Text snippet — RU-демо с триггером, целью, плейсхолдерами (вместо EN про «посылку»).

### Навигация и таблицы

- `00-index.md` — блок Дополнительно с короткими пояснениями; порядок: Отзывы → Email → Заготовки → Ссылки → Безопасность.
- `INDEX.md` — подписи RU / «демо».
- `Other--d2ab6b95.csv` (+ all) — описания в духе мини-гайдов; Name «Текстовые заготовки».
- Трек «Развитие» — чуть усилены формулировки у Отзывов/Ссылок.
- `docs/glossary.md` — термины «Текстовые заготовки», «Дополнительно (Other)».

## Методология

- **Cagan:** каждый блок ведёт к outcome (обучение, доверие, скорость ответа, меньше риска), не к «заполнить таблицу».
- **Mom Test (лёгкий слой в Отзывах):** сильный vs слабый сигнал; похвала ≠ валидация. Полная перестройка CSV клиентов — всё ещё пакет **E**.
- **Ries:** шаблоны и заготовки ускоряют delivery; отзывы и ссылки кормят learning — не путать.
- Секреты / PII — отсылки к «Цифровая безопасность», без хранения секретов в шаблоне.

## Перенос в Notion (вручную)

1. Обновить текст страниц: Отзывы, Email шаблоны, Text Snippets → заголовок «Текстовые заготовки», Ссылки.
2. Обновить демо-записи Alice / Thank you… / Thanks for reaching… (или заменить своими).
3. В таблице раздела Other — Description как в CSV; переименовать пункт Text Snippets в UI.
4. Views баз можно не менять (схема колонок Feedback/Links почти та же; расширение полей отзывов — опционально или с пакетом E).

## Вне скоупа

- Полный Mom Test + схема DB клиентов/отзывов (пакет E).
- Rename папок `Text Snippets/` / hash-файлов.
- Библиотека знаний, лендинг.
