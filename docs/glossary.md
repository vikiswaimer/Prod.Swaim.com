# Глоссарий Prod.Swaim (пакет C)

Единые термины для шаблона и агента. Тон текстов: **«вы»**. Язык интерфейса шаблона: **RU** (EN — только осознанный термин в скобках).

| Термин | Как пишем | Смысл |
|--------|-----------|--------|
| Бренд | **Бренд** (не «Брэнд») | Позиционирование + визуальный/голосовой стиль. Папка в Git пока может называться `Брэнд/` — в текстах всегда «Бренд». |
| Активности | **Активности** (не «Aктивности» с лат. A) | Ежедневный ритм: задачи, идеи, цели… |
| Стратегия маркетинга | страница `Бизнес/Маркетинг` | Куда и зачем: сегмент, обещание, канал, гипотеза |
| Маркетинг (операции) | модуль `Маркетинг/` | Планировщики: кампании, посты, статьи, email |
| Кампания | **кампания** (не «компания») | Маркетинговая/продуктовая кампания |
| MVP | MVP | Уже **полезный** (и по возможности платящий) рабочий срез продукта |
| Гипотеза | гипотеза | Проверяемое утверждение о клиенте/канале/ценности |
| Outcome | исход / результат | Изменение поведения или метрики, не «сделали фичу» |
| Руководство по стилю | предпочтительно так | EN «Style Guide» допустим как подпись файла до переименования |

## Дубли, которые нельзя смешивать

1. **Стратегия маркетинга** (`Бизнес/…`) ≠ **операционный маркетинг** (`Маркетинг/…`).
2. **Идея** ≠ **задача** ≠ **цель**.
3. **Пример** (`examples/` или пометка) ≠ чистый шаблон для покупателя.
